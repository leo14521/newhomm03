# 조선일보명조체 (본문 폰트)

미술관 컨셉 본문에 사용하는 한글 명조체입니다.

1. **다운로드**: [조선일보 100주년 전용서체](https://event.chosun.com/100/100font.html) 또는 [ChosunNm 웹폰트(woff)](https://event.chosun.com/100/100font.html)에서 조선일보명조체를 받습니다.
2. **woff 파일**을 `Chosunilbo_myungjo.woff` 이름으로 이 폴더(`public/fonts/`)에 넣어 주세요.

파일이 없으면 사이트는 기본 폰트(Noto Sans KR)로 표시됩니다.
