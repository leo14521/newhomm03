## UX 및 디자인 구조 분석

### 더안청담의원 UX 특징
1. **히어로 섹션**: "From the inside, from the AaN!" 메시지와 함께 다크 배경에 골드 액센트
2. **프리미엄 리프팅 섹션**: 3개 카드 레이아웃 (울쎄라, 써마지, 온다) - 각각 로고 이미지와 설명
3. **베스트 프로그램**: 슬라이더 형태로 프로그램 소개
4. **전후사진 갤러리**: 그리드 레이아웃, 로그인 후 확인 가능
5. **인테리어 갤러리**: 수평 스크롤 형태

### BLS클리닉 UX 특징
1. **상단 정보 카드**: 시술시간, 마취여부, 회복기간, 유지기간을 아이콘과 함께 표시
2. **원리 설명**: 3단계 애니메이션 형태 (시술 전/중/후)
3. **효과 섹션**: 3가지 효과를 아이콘과 함께 카드 형태로 표시
4. **시술 가능 부위**: 인터랙티브 맵 형태, 클릭 시 해당 부위 강조
5. **시술 과정**: 8단계 스텝 표시
6. **Q&A 아코디언**: 질문 클릭 시 답변 펼쳐지는 형태
7. **BLS 약속**: 6개 약속을 이미지와 함께 그리드 레이아웃

### 파랑새피부과 UX 특징
1. **히어로 섹션**: "당신이 몰랐던 당신의 아름다움" 큰 타이틀
2. **원장 메시지**: 원장 사진과 함께 개인적인 메시지
3. **피부 재생 원리**: GIF 애니메이션으로 콜라겐 재생 과정 시각화
4. **초음파 에너지 설명**: 다이어그램과 함께 팁 종류별 설명
5. **상담 사례**: 필터 기능이 있는 갤러리 (부위별, 연령대별 필터)
6. **Q&A 섹션**: 질문과 답변을 카드 형태로 표시

### 오드의원 UX 특징
1. **칼럼 형식**: 원장이 직접 작성한 상세한 설명 글
2. **이미지 삽입**: 원리 설명, 비교 이미지, 차트 등이 글 중간에 삽입
3. **단계별 설명**: STEP 1, STEP 2 형태로 명확하게 구분
4. **표 형태**: 샷수별 가이드라인을 표로 정리

## 주요 클리닉 및 시술

### 더안청담의원
- 안티에이징센터에서 울쎄라, 써마지, APTOS 바이오리프팅, 실루엣 소프트 등 다양한 리프팅 시술 제공
- 인터널 안티에이징센터 운영
- 콜라겐센터 운영

### BLS클리닉
- 울쎄라 정품팁 인증서 발급
- 실시간 초음파 영상기술(DeepSee™) 사용
- 6가지 약속 시스템 (편히 쉬세요, 걱정마세요 등)

### 파랑새피부과
- 디자인노트 시스템으로 재시술 시 맞춤 시술
- 원장 직접 시술 및 상담
- 다양한 초음파 에너지로 맞춤 시술

### 오드의원
- 원장 칼럼을 통한 상세한 정보 제공
- 1:1 상담을 통한 맞춤 플랜
- 다양한 장비 조합 제안

## 적용할 UX 패턴

1. **히어로 섹션**: 큰 타이틀 + 서브 타이틀 + 배경 이미지 (다크 오버레이)
2. **원리 설명**: 3단계 애니메이션 또는 카드 형태
3. **효과 섹션**: 아이콘 + 텍스트 카드 그리드
4. **시술 과정**: 스텝 형태의 타임라인
5. **Q&A**: 아코디언 또는 카드 형태
6. **전후사진**: 그리드 레이아웃 또는 슬라이더
7. **인터랙티브 요소**: 부위 선택 맵, 필터 기능
